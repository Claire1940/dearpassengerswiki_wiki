<Section icon={Target} title="初心者ガイド：初フライトを成功させるために">

**Dear Passengers VR**で成功を収めるには、単に飛行機を空に浮かせておくだけでは不十分です。すべてのフライトは、利益と生存の微妙なバランスの上に成り立っています。初心者は、フライト前計画から乱気流後の復旧まで続く「フライトループ」を理解することに集中すべきです。

主な目的は、乗客と貨物の両方を、最小限の損害で目的地まで届けることです。ゲームは物理駆動エンジンを使用しているため、パイロットによる急旋回や、客室内のロックされていないスーツケース一つ一つが、飛行の安全性に直接的な影響を及ぼします。

<Steps>
  <Step num="1" title="搭乗名簿（マニフェスト）の選択">
    離陸前に、乗客と貨物を選ぶ必要があります。報酬が高いほど「気難しい乗客」や「違法な貨物」が含まれ、飛行中のトラブル発生率が高まります。
  </Step>
  <Step num="2" title="役割分担">
    チームを分割します。常に少なくとも一人がコックピットにいる必要があり、残りのクルーは客室を管理します。突然のエアポケットを生き延びるにはコミュニケーションが不可欠です。
  </Step>
  <Step num="3" title="サービスとセキュリティ">
    離陸後は、客室クルーが飲食を提供して乗客の満足度を維持します。同時に、機体操作中に貨物が固定されているかを常に確認しなければなりません。
  </Step>
  <Step num="4" title="緊急対応">
    エンジンの火災や動物の脱走などの災害が発生した場合は、直ちにすべてのサービスを停止してください。消火器や拘束具を使用して、機体が損傷する前に脅威を抑え込みます。
  </Step>
</Steps>

<Callout type="warning" title="物理演算の警告">

<Section icon={Shield} title="乗客と貨物のリスク管理">

**Dear Passengers VR**の経済システムは、リスクと報酬のバランスに基づいています。フライト名簿を選択することは、実質的に難易度を選択することと同じです。標準的な乗客と通常の貨物を選べば安定した安全な収入が得られますが、本格的なアップグレードは最も危険な任務の先に用意されています。

| カテゴリ | リスクレベル | 報酬 | 主な危険要因 |
| :--- | :--- | :--- | :--- |
| **標準的な乗客** | 低 | 基本 | 些細なサービスへの不満 |
| **気難しい乗客** | 中 | 高 | 頻繁な要求 / 注意力の散漫 |
| **手に負えない乗客** | 高 | 非常に高 | 物理的な混乱 / 妨害行為 |
| **通常の貨物** | 低 | 基本 | 旋回時の重心移動 |
| **危険な貨物** | 中 | 高 | 漏洩や火災の可能性 |
| **生きた動物の貨物** | 極限 | プレミアム | ワニの脱走 / 負傷 |
| **違法な貨物** | 高 | 極限 | 高強度の飛行トラブル |

<Callout type="info" title="エキスパートの戦略">
新しいクルーとプレイする場合は、「極限」レベルの動物貨物は避けましょう。クルーが拘束メカニクスに慣れていないと、脱走した捕食者によって数秒でフライトが終了する可能性があります。
</Callout>

</Section>

<Section icon={Activity} title="飛行トラブルと緊急プロトコル">

**Dear Passengers VR**において、環境は最大の敵です。ダイナミックな天候は晴天を一瞬にして悪夢に変え、機械の故障は即座の修理を必要とします。トラブルの優先順位を理解することが、長期的な進行において重要です。

**一般的なトラブルと対応：**
- **ダイナミックな天候**: パイロットは高度と速度を調整する必要があります。客室クルーはすべてのカートを固定してください。
- **乱気流**: 物理演算によりプレイヤーや物体が投げ飛ばされます。怪我を避けるため、しゃがむか固定された構造物に掴まってください。
- **バードストライク**: エンジン故障や機体の亀裂を引き起こす可能性があります。クルーの一人が客室の外に出て外部修理を行う必要があります。
- **機内火災**: 客室内に急速に広がります。消火器の即時使用と乗客の避難が必要です。

| トラブルの種類 | 最初の対応者 | 重要なアクション | 悪化のリスク |
| :--- | :--- | :--- | :--- |
| **エンジン火災** | パイロット/技術者 | 燃料遮断 / 消火 | 制御の完全喪失 |
| **機体の亀裂** | 緊急技術者 | 外部修理 | 減圧 / 乗客の吸い出し |
| **貨物の脱走** | 貨物監視員 | 封じ込め / 拘束 | クルーの負傷 / サービス停止 |
| **暴徒化** | 客室責任者 | 沈静化 | 客室全体の混乱 |

<Callout type="warning" title="機体の完全性">
機体に亀裂が生じた場合、パイロットは直ちに安全な高度まで下降しなければなりません。これを怠ると、乗客の健康状態が急速に悪化します。
</Callout>

</Section>
