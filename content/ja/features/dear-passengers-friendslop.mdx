<Section icon={Swords} title="フライトオペレーションと混沌とした機内メカニクス">

ゲームの核心は、非常に不安定な貨物機を墜落させずに飛行させ続けることです。従来のフライトシミュレーターとは異なり、コックピットの操作は簡略化されていますが、飛行機を取り巻く環境は敵対的で予測不可能です。プレイヤーは、操縦桿から後部のカーゴランプに至るまで、機体のほぼすべての部分を手動で操作する必要があります。機体との物理的なインタラクションは非常にダイナミックで、高高度を飛行中にサイドドアを開けることができ、機内を猛烈な風圧にさらすことになります。

航空機の管理は、単に機首を前に向けておくだけではありません。客室と外側の主翼がアクティブなプレイエリアとなります。エンジンが故障したり火災が発生したりした場合は、プレイヤーは物理的に機外に出て、主翼の上に立って手動で修理を行う必要があります。これには落下するリスクが伴うため、慎重な動きとタイミングが必要です。さらに、不安定な貨物を管理し、乗り込もうとする敵から機体を守り、戦闘地帯をナビゲートしなければなりません。

| メカニクス | 説明 | カオス度 |
| :--- | :--- | :--- |
| 主翼立ち | エンジンの修理や戦闘のため、飛行中に機外に出て歩く | 高 |
| 貨物管理 | ワニや重い木箱などの危険なアイテムを輸送する | 中 |
| キャビン防衛 | 侵入者に対し、AK-47などの武器を使用して機内を守る | 高 |
| 緊急脱出 | 突然のエンジン爆発や飛行中の構造的崩壊への対処 | 致命的 |

銃器の導入は、物理ベースのパーティーシミュレーターでは珍しい防衛プレイのレイヤーを加えています。プレイヤーはAK-47などの武器を装備して、迎撃してくる敵機を撃墜したり、貨物室に乗り込もうとする敵対勢力を排除したりできます。メンテナンス、ナビゲーション、そしてアクティブな戦闘の間を絶え間なく移行するこのプロセスは、常に注意を怠れない緊迫したゲームプレイループを生み出します。
