<Section icon={Target} title="協力フライトのセットアップガイド">

**Dear Passengersのローカル協力プレイ**でセッションを開始するには、特定のプロセスを踏む必要があります。ただコックピットに乗り込むだけでなく、ルートを計画し、装備を選択しなければなりません。報酬が高ければ高いほど、フライトは混沌としたものになります。

<Steps>
  <Step num="1" title="ロビーの作成">
    Steamインターフェースを介してセッションをホストします。フレンドを招待し、全員が物理演算アクションに適した好みの入力デバイスを設定していることを確認します。
  </Step>
  <Step num="2" title="契約（ルート）の選択">
    利用可能なフライト契約を確認します。各契約には、乗客数、貨物の種類、予想される気象条件が記載されています。最初の数フライトは「スタンダード」難易度から始めることをお勧めします。
  </Step>
  <Step num="3" title="役割の割り当て">
    「開始」をクリックする前に、誰がパイロットをやるかを決めます。これが最も重要な役割です。また、貨物スペシャリストには、輸送箱の物理挙動を理解している人を任命してください。
  </Step>
  <Step num="4" title="飛行前チェック">
    客室を歩き回ってみましょう。消火器の位置を確認し、リスクの高い貨物コンテナを特定します。機内が停電する前に、レイアウトを把握しておくことが不可欠です。
  </Step>
  <Step num="5" title="離陸と安定化">
    パイロットが上昇を開始したら、客室乗務員はすぐに固定されていないオブジェクトの確保を開始してください。乱気流は早い段階で発生することが多く、固定されていないものはすべて凶器と化します。
  </Step>
</Steps>

<Callout type="success" title="戦略の勝利">
離陸前に60秒間かけて「緊急時の配置」を計画したチームは、バードストライクやエンジン故障を生き延びる確率が40%向上します。
</Callout>

</Section>

<Section icon={Shield} title="チームでの機内ハザードへの対処">

**Dear Passengersのローカル協力プレイ**の醍醐味は、災害の管理です。ゲームには動的なハザードシステムが採用されており、「こぼれた飲み物」のような小さな問題でも、適切に対処しなければ機体外壁の破壊へとエスカレートする可能性があります。

| ハザード | 発生トリガー | チームの対応策 |
| :--- | :--- | :--- |
| **乱気流** | 天候 / パイロットのミス | パイロットは機体を水平に保つ。乗務員は手すりや座席にしがみつく |
| **エンジン火災** | バードストライク / オーバーヒート | パイロットは高度を維持。乗務員は消火器を使用 |
| **貨物の脱走** | 物理的な衝撃 / 破損 | 貨物担当が捕獲。サポートは乗客を避難させる |
| **暴れる乗客** | 満足度の低下 | サービス担当が拘束具や気晴らし用のアイテムを使用 |
| **外壁の破損** | 外部からの衝撃 | 1人のプレイヤーが修理。他のメンバーは客室の気圧を維持 |

これらのハザードに対処するには「優先順位の判断（トリアージ）」が必要です。何がフライトを終わらせる原因になりやすいかを見極めなければなりません。エンジン火災は怒れる乗客よりも危険です。貨物室のワニの脱走は、こぼれたコーヒーよりもはるかに危険です。

<Section icon={Sparkles} title="乗客と貨物のリスク管理">

**Dear Passengersのローカル協力プレイ**では、取るリスクの大きさによって収入が決まります。「一般乗客」を選べば安定した収入が得られますが、額は低いです。「違法貨物」や「危険な動物」を選ぶと莫大な報酬が得られますが、全損する確率が大幅に上昇します。

| カテゴリ | リスクレベル | 報酬 | 主な課題 |
| :--- | :--- | :--- | :--- |
| **一般乗客** | 低 | 低 | ルーチンワーク、退屈への対処 |
| **VIP乗客** | 中 | 高 | 要求が厳しく、特別なニーズがある |
| **乱暴な集団** | 高 | 極めて高 | 頻繁に発生する喧嘩、客室の破損 |
| **一般貨物** | 低 | 低 | 重量バランスの管理 |
| **生きた動物** | 極限 | 莫大 | 脱走、怪我のリスク |
| **違法物資** | 高 | 高 | 外部からの脅威、極端な重量 |

4人のフルチームでプレイする場合は、「混合高リスク」フライトにも挑戦する余裕があります。人数が少ない場合は、暴動に対応しながら爬虫類と格闘する人手がないため、価値の高い貨物と低リスクの乗客の組み合わせに留めることをお勧めします。

<Callout type="tip" title="プロの戦略">
「動物貨物」を運ぶ場合は、フライト中ずっと1人のプレイヤーを貨物室に常駐させてください。箱が壊れる前にキャッチする方が、逃げ出した後に動物を追いかけるよりもはるかに簡単です。
