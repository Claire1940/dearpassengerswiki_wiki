<Section icon={Sparkles} title="コアゲームプレイメカニクスと特徴">

**Dear Passengers**のメカニクスを理解しているかどうかが、無事な着陸と炎上墜落の分かれ目となります。ゲームは主に「空港フェーズ」と「飛行フェーズ」の2つのフェーズに分かれており、それぞれ異なるスキルと協力レベルが求められます。

| ゲームプレイフェーズ | 主な目的 | 主な課題 |
| :--- | :--- | :--- |
| **空港フェーズ** | チェックイン、保安検査、搭乗 | 武装NPC、手荷物のカオス、制限時間 |
| **飛行フェーズ** | 操縦、乗客の世話、修理 | バードストライク、エンジン故障、乗客の怒り |
| **戦闘フェーズ** | 機体の防衛 | 敵機、空中乗り込み、パラシュート降下 |

空港フェーズでは、混雑したターミナルを進まなければなりません。ここは普通の空港ではありません。警備員がAK-47で武装していることもあれば、乗客が暴れることも日常茶飯事です。タイマーがゼロになる前に全員を正しい飛行機に乗せるためには、連携が鍵となります。

一度空に飛び立つと、ゲームは慌ただしい管理シミュレーションへと変化します。ただ飛行機を操縦するだけでなく、水漏れを直したり、エンジンを再起動したり、時には問題のある乗客を非常口から投げ捨てたりします。物理エンジンにより、飛行機のあらゆる動きが機内のプレイヤーに影響を与えるため、客室内で爆笑の「玉突き衝突」が発生します。

<Section icon={Target} title="ハザードとスペシャルイベントの攻略">

2026年のゲームシーンにおいて**Dear Passengers**を際立たせているのは、その奇妙なハザードの数々です。これらは単なる障害物ではなく、ゲームセッションの軌道を完全に変えてしまうスクリプトイベントです。政治的なパロディから環境災害まで、あらゆる事態に備える必要があります。

| ハザードタイプ | 説明 | 生存戦略 |
| :--- | :--- | :--- |
| **バードストライク** | 鳥の群れがエンジンに激突する | 回避するか、すぐに修理する |
| **VIPカオス** | 特殊なNPC（「トランプ」のパロディなど） | 機嫌を取るか、機外へ排出する |
| **武装警備員** | AK-47を持った空港の警備員 | プロトコルに従うか、全力で走る |
| **空中戦** | 敵機からの攻撃を防ぐ | 客室内のアイテムを投擲物として使用する |

「VIP」乗客は特に厄介です。最近のゲームプレイトレーラーで見られるように、これらのキャラクターは客室内に大混乱を引き起こします。一部のプレイヤーは、これらのキャラクターを「平手打ち」して飛行機から投げ捨てることを選択します。これは目の前の問題を解決しますが、ラウンド終了時のスコアが下がる可能性があります。

<Callout type="info" title="プロの洞察">
このゲームは「クリエイティブ」な問題解決を評価します。エンジンが火を吹いていて消火器がない場合、乗客の水筒（あるいは乗客自身）を投げつけることで、予想外の結果が得られることがあります。
</Callout>
