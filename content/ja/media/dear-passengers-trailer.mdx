<Section icon={Map} title="明らかになった主要なゲームプレイ要素">

**『Dear Passengers』のトレーラー**では、プレイヤーが遭遇する主要なシステムが詳しく紹介されています。従来のフライトシミュレーターとは異なり、本作は危機状況下における航空機の内部および外部の管理に焦点を当てています。物理演算に基づいたインタラクションにより、プレイヤーは飛行中の飛行機の翼の上に立ったり、飛行機間で飛び移ったりすることさえ可能です。

| 特徴 | 説明 | カオス度 |
| :--- | :--- | :--- |
| **手動飛行** | 航空機の飛行ルートと高度を直接制御する | 中 |
| **客室気圧** | 飛行中にドアを開けると、安定性とプレイヤーの安全性に影響する | 高 |
| **貨物管理** | アリゲーターや不安定な木箱などのアイテムを輸送する | 高 |
| **ゲーム内チャット** | 連携のための近接型音声コミュニケーション | 可変 |
| **戦闘システム** | 火器を使用して飛行機を防衛する、または他を攻撃する | 極めて高 |

トレーラーでは、カオスな協力プレイゲームを揶揄する言葉として一部で使われる「Friend Slop（フレンドスロップ）」論争に特に焦点を当てています。しかし、飛行中の飛行機間で貨物を移動させる機能など、示されたメカニズムの深さは、単なる「スロップ（駄作）」を超えた完成度を示唆しています。

<Callout type="info" title="メカニズムの深さ">
ゲームはカオスに見えますが、トレーラーは高速で飛行する機内環境において、プレイヤーやオブジェクトがどのように動くかを制御する複雑な物理システムが存在することを明かしています。
</Callout>
