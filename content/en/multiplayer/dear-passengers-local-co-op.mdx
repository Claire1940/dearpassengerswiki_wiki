<Section icon={Shield} title="Managing In-Flight Hazards as a Team">

The core of **dear passengers local co op** is managing disasters. The game uses a dynamic hazard system where one small problem—like a spilled drink—can escalate into a hull breach if not handled correctly.

| Hazard | Trigger | Team Response |
| :--- | :--- | :--- |
| **Turbulence** | Weather / Pilot error | Pilot levels out; Crew holds onto rails or seats |
| **Engine Fire** | Bird strike / Overheat | Pilot maintains altitude; Crew uses extinguishers |
| **Escaped Cargo** | Physics / Damage | Cargo lead contains; Support clears passengers |
| **Unruly Passenger** | Low satisfaction | Service crew uses restraints or distractions |
| **Hull Breach** | External attack | One player repairs; Others maintain cabin pressure |

Managing these hazards requires "Triage." You must decide what is most likely to end the flight. A fire in the engine is more dangerous than an angry passenger. A loose crocodile in the cargo hold is more dangerous than a spilled coffee.
